Welcome

This is the webiste created for week 8 of cs50. The purpose is to show what we learned in HTML, CSS and Javascript, along some other topics. I am trying to replicate the original cs50 website by looking directly at the elements tab inside developer tools. I am doing this as practice to understand how a website is structured. I think the format of having a side bar with the control menu and the main page next to it is a simple yet extremely functional way to present information.

Please navigate through the left menu to visit the different pages. Home will redirect you to this page. About has some information about me. Blog has some place for personal thoughts, and Projects will be a place to include personal projects.

Still looking into bootstrap and how it works. Another good idea would be to create another page that includes the things I learned creating this website, where I explain the challenges and solutions I came up with.
